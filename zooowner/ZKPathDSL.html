trait ZKPathDSL extends AnyRef

Provides utilities for constructing and pattern matching [ZKPath]s.

You can either extend class where you intend to use path DSL to make it available only inside that class or use global import import zooowner.ZKPathDSL._

After enabling path DSL you can:

Construct paths:

// $ is an alias for path root.
val path = $ / "foo" / "bar"

Deconstruct paths:

ZKPath("/foo/bar") match {
 // parent = ZKPath("/foo"), child = "bar"
 case parent/child =>
 // foo = "foo", bar = "bar"
 case $ /foo/bar =>
 // foo = "foo", rest = ZKPath("/bar")
 case foo/:rest =>
 // foo = "foo", bar = "bar"
 case foo/:bar/: $ =>

You can also match strings:

val $ /foo/bar = "/foo/bar"
